# 🎰 Python Slot Machine
## 🎯 Project Goal
This project simulates a realistic casino slot machine with GUI, paylines, and betting logic.

A realistic slot machine built with Python and Tkinter.

## 🧠 Features
- GUI interface (Tkinter)
- Multiple paylines (rows + diagonals)
- Animated spinning
- Win highlighting
- Betting system with balance tracking
- Deposit system

## 🎮 How to run

```bash
python3 main.py
